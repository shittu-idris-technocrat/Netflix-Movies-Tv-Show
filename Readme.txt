📊 Netflix Movies & TV Shows Analysis
This project analyzes a dataset of Netflix titles to uncover trends and insights, using Python and Pandas.

📌 Key Insights
Netflix has more movies than TV shows.
Most content comes from the US, India, and UK.
A significant spike in content occurred post-2016 and starting declining from 2020
Most frequent genres include Dramas, Comedies, and Documentaries.
Director with the most titles: [Raul Campos][Jan Suter].
📁 Tools Used
Python (Pandas, Matplotlib)
Jupyter Notebook
Dataset: Netflix on Kaggle
